## Run Locally

Clone the project

```bash
  git clone https://github.com/kabhinav577/react-chat-engine-app.git
```

Go to the project directory

```bash
  cd react-chat-engine-app
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```

## Tech Stack

**Client:** Vite , React JS, ChatEngineUI SDKs
